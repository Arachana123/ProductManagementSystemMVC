# Product Management System (ASP.NET Core MVC)

This is a Product Management System built using ASP.NET Core MVC and SQL Server.

## Features
- Add Product
- Edit Product
- Delete Product
- Product Details Page
- Image Upload
- Live Search
- Pagination
- Bootstrap Card UI

## Technologies Used
- ASP.NET Core MVC
- C#
- SQL Server
- Bootstrap
- Entity Framework
